# neotest

## Test Summary Keymaps

| key binding       | descriptions         |
| ----------------- | -------------------- |
| `?`               | show help            |
| **summary panel** |                      |
| `<CR>`            | expand               |
| `e`               | expand all           |
| `i`               | jump to test         |
| `K`               | previous failed test |
| `J`               | next failed test     |
| **test actions**  |                      |
| `r`               | run                  |
| `u`               | stop                 |
| `d`               | debug                |
| `w`               | watch                |
| `a`               | attach               |
| **test output**   |                      |
| `o`               | output               |
| `O`               | shortened output     |
| **mark**          |                      |
| `m`               | mark                 |
| `M`               | clear marks          |
| `R`               | run marked           |
| `D`               | debug marked         |
| **target**        |                      |
| `t`               | target               |
| `T`               | clear target         |
