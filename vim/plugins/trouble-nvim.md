# trouble.nvim

| key binding          | descriptions                            |
| -------------------- | --------------------------------------- |
| `?`                  | show help                               |
| `q`                  | quit                                    |
| **refresh**          |                                         |
| `r`                  | refresh                                 |
| `R`                  | toggle refresh                          |
| **jump**             |                                         |
| `<Esc>`              | cancel and go back to original position |
| `<CR>`               | jump                                    |
| `o`                  | jump and close                          |
| `<C-s>`              | open with horizontal split              |
| `<C-v>`              | open with vertical split                |
| **navigate in file** |                                         |
| `}`/`]]`             | next item                               |
| `{`/`[[`             | previous item                           |
| **preview**          |                                         |
| `i`                  | inspect                                 |
| `p`                  | preview                                 |
| `P`                  | toggle preview                          |
| **delete entry**     |                                         |
| `dd`                 | delete entry                            |
| `d`                  | delete entry in VISUAL mode             |
| **fold actions**     |                                         |
| `zo`                 | open fold                               |
| `zO`                 | recursive open fold                     |
| `zc`                 | close fold                              |
| `zC`                 | recursive close fold                    |
| `za`                 | toggle fold                             |
| `zA`                 | recursive toggle fold                   |
| `zm`                 | decrease fold level                     |
| `zM`                 | close all folds                         |
| `zr`                 | increase fold level                     |
| `zR`                 | open all folds                          |
| `zx`                 | update fold                             |
| `zX`                 | update all folds                        |
| `zn`                 | disable fold                            |
| `zN`                 | enable fold                             |
| `zi`                 | toggle fold enable                      |
